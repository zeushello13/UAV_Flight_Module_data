# Система управления данными для мониторинга полётов БПЛА

## Описание проекта

Данный репозиторий содержит прототип информационной системы управления данными,
предназначенной для мониторинга, анализа и хранения полётных данных беспилотных
летательных аппаратов (БПЛА).

Система ориентирована на работу с телеметрией, логами полётов и событиями отказов.
Основной акцент сделан на централизованное хранение данных, автоматическую
валидацию, формирование отчётов и удобный пользовательский интерфейс для анализа
результатов полётов.

---

## Основной функционал

- импорт и структурированное хранение полётных данных;
- централизованная база данных PostgreSQL для временных рядов и метаданных;
- SQL-логика на стороне БД (триггеры, функции, процедуры);
- автоматическая валидация телеметрии и событий отказов;
- формирование агрегированных отчётов по полётам;
- клиентское графическое приложение (GUI) на Python (Tkinter);
- визуализация телеметрии и анализ отказов;
- экспорт данных и отчётов в CSV и JSON.

---

## Архитектура системы

Система включает следующие основные компоненты:

- подсистема импорта полётных данных;
- централизованная база данных PostgreSQL;
- SQL-логика (триггеры, функции, процедуры);
- клиентское приложение с графическим интерфейсом;
- подсистема формирования отчётов и диагностики.

Значительная часть логики реализована на уровне базы данных, что обеспечивает
целостность, предсказуемость и автоматизацию обработки данных.

---

## Используемые технологии

- **Язык программирования:** Python  
- **GUI:** Tkinter  
- **База данных:** PostgreSQL  
- **Доступ к БД:** psycopg2  
- **Визуализация:** matplotlib
